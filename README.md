# PNY Consulting

Made with Hugo (Go-based static site generator) for [pnyconsulting.com](https://www.pnyconsulting.com).

## Development Server

To run the development server, use:

`$ hugo server -D`

And then navigate to http://localhost:1313/.

## Building for Production

`$ hugo -D`
